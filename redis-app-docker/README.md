Hoy terminé un pequeño proyecto para practicar arquitectura distribuida usando Docker y Python.

Construí una API con FastAPI que envía tareas a una cola Redis y las procesa con workers en segundo plano.

Tecnologías usadas:
• FastAPI
• Redis Queue (RQ)
• Docker
• Docker Compose
• Docker Swarm

Este tipo de arquitectura se usa mucho para procesar tareas pesadas en sistemas reales (emails, procesamiento de imágenes, pipelines de datos, etc.).
